<h2 id="duckgu-details">🎬 덕구(덕질친구를 구해요) 상세</h2>

#### ✔️ 프로젝트 개요

**목표:** 인간은 '관계'를 중요하게 여긴다는 점에 착안하여 이를 기반으로 한 영화 추천 서비스 개발
<br>
**배경:** 취향이 비슷한 사용자들이 있는 영화 모임(Meet-둥지) 추천 애플리케이션 기획

#### ✔️ 주요 기능

**Keyword:** 개인 취향을 파악하기 위한 키워드
<br>
**Meet(둥지):** 영화 취향이 맞는 사람들이 모인 모임
<br>
**Review:** 영화에 대한 감상
<br>
**Movie:** 영화 상세 페이지에선 영화 정보 및 리뷰 제공
<br>
**User:** 개인 별 선호 키워드 설정 가능

#### ✔️ 담당 역할 및 기여

**역할:** 팀장, FrontEnd 개발, UX/UI 디자인, BackEnd 개발 일부
<br>
**기여:** users fixture 생성, 회원가입, 로그인, 로그아웃 기능 구현, 홈, 회원가입, 로그인, 모임 필터링, 모임 상세, 영화 정보, 영화 그룹 페이지 생성

#### ✔️ 사용 기술/이유

**사용 기술:** Django REST Framework + Vue.js
<br>
**선택 이유:** 빠르고 안정적인 개발을 위해 선택한 프레임워크
